# 🛡️ FraudGuard Mobile - Demo App

Aplicativo Android de demonstração para o modelo de detecção de fraudes bancárias.

## 🛠️ Configuração Inicial

1. **Instalar Dependências:**
   ```bash
   cd fraud_detection_mobile
   npm install
   ```

2. **IP da API:**
   - O App está configurado para o IP: `192.168.0.31:8000`.
   - Se o IP do seu PC mudar, altere em `screens/HomeScreen.js`.

## 🚀 Como Executar

### Usando o Expo Go (Desenvolvimento)
1. Instale o app **Expo Go** na Google Play Store.
2. PC e Celular devem estar no **mesmo Wi-Fi**.
3. No terminal:
   ```bash
   npx expo start
   ```
4. Escaneie o QR Code com o Expo Go.
